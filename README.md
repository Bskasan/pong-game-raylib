# Pong

A classic Pong game built with [raylib](https://www.raylib.com/) in C/C++.

## Features

- Single-player mode vs AI
- Local 2-player mode
- Pause menu
- Score tracking

## Controls

**Main menu**
- `Enter` — Play vs AI
- `P` — 2-player mode
- `Esc` — Quit

**In game**
- `Up` / `Down` — Right paddle
- `W` / `S` — Left paddle (2-player only)
- `Space` — Pause

**Paused**
- `P` — Resume
- `Esc` — Back to menu

## Build

Requires a C++17 compiler and raylib.

### Windows (MSVC) Build

```
- Don't forget to update your project path.
build.bat
```
